BIENVENUE SUR LE JEU DE LA BATAILLE NAVALE

La bataille navale est un jeu de stratégie au tour par tour.
Dans cette version, chaque joueurs disposent de 10 bâteaux :
		
		- 5 frégates (nécessite 1 coup pour être détruit)
		- 2 destroyers (nécessite 2 coup pour être détruit)
		- 2 torpilleurs (nécessite 3 coup pour être détruit)
		- 1 porte-avions (nécessite 5 coup pour être détruit)
		
Le jeu se déroule de la manière suivante :

En premier lieu, le joueur 1 positionne ses bateaux, il suffit de rentrer la case correspondante comme ceci :
	"Ou voulez vous positioner vos 5 frégates ?"  "B6, B1, A4, C2, A9, D2"
	
Une fois que le premier joueur a positionné ses bateaux, le joueur 2 positionne les siens.
Lorsque les joueurs ont fini de positionner leur bateaux, le jeu commence.

L'ordinateur demande donc au premier joueur une case à attaquer, il suffit de rentrer la case correspondante comme ceci :
	"Ou voulez vous attaquer ?"  "B4"
	
Si la case contient un bateau adverse, le message suivant s'affichera "Bateau touché !"
Si un joueur détruit un bateau ennemi, l'ordinateur affichera "Bateau coulé ! 9 bâteaux restants !"
Si à l'inverse la case ne contient pas de bateaux, le message sera le suivant "Aucun bateau touché, au joueur suivant !"
Tant que le joueur réussi ses attaques, il garde la main.
Si il ne touche pas de bateaux, le joueur adverse prend la main.

Le jeu se finit lorsqu'un joueur n'a plus de bateaux.




PS : jeu non finit
